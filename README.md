## 使用 mybatis-01 说明

下载使用需要重命名 `mybatis-config.default.xml` 文件为 `mybatis-config.xml` ， 同时将里面的参数改成自己的数据库连接

设置连接时建议增加 `useSSL=false&amp;useUnicode=true&amp;characterEncoding=UTF-8` 参数
